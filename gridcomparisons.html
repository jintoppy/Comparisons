Yes Cut, Copy, Paste using standard shortcuts ( Control + X, Control + C, Control + V) are supported. However, doing these actions using a context menu is not supported. If we specify a height to the table(which will enable scrollbar for excel), it is supporting 5000 rows. But, with 1 million rows (maximum possible), the page is broken. If we don't specify a height, even 5000 rows is not supported. In short, 5000 rows are also supported.
